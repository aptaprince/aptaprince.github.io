Pengalamanku Menggunakan VPS dari Indocenter
Cerita pengalamanku mencoba layanan VPS dari Indocenter selama 3 Bulan.
Virtual Private Server (VPS) adalah server pribadi yang keseluruhan resource-nya hanya digunakan oleh satu pengguna saja dan tidak dipengaruhi oleh pengguna lain. Pengguna dapat mengelola secara penuh semua konfigurasi dan resource yang ada pada VPS dan melakukan apa pun yang diinginkan.
Teknologi yang digunakan VPS adalah virtualisasi hardware server fisik yang kemudian dibagi menjadi beberapa resource berbeda. Disebut virtual karena pembagian ini dilakukan dengan menggunakan software sehingga dalam satu server fisik bisa terdapat beberapa VPS yang dijalankan.
Source: niagahoster.co.id
Halo teman-teman semua, jadi pada kesempatan kali ini aku mau share pengalamanku setelah mencoba layanan VPS dari Indocenter nih!
Jadi aku berkesempatan untuk mencoba layanan trial VPS dari Indocenter selama 3 bulan setelah menang kuis di salah satu webinar dari KLAS (Kelompok Linux Arek Suroboyo), hehe lumayan kan bisa dapetin kesempatan buat cobain VPS selama 3 bulan.
indocenter.co.id
Bisa dibilang ini pengalaman pertama kali buat aku untuk nyobain pake VPS ini. Sebelumnya kalau belajar system administration seringnya sih lewat virtual machine aja, hehehe..
Lalu, buat spesifikasi VPS yang aku coba kali ini yaitu Dual Core CPU dan 1 GB RAM, sangat cukup buat coba-coba belajar bikin server. Jadi untuk akses VPS dari Indocenter ini aku pakai Software Remote SSH, yaitu PuTTY. Hal pertama kali yang aku lakukan setelah hands-on VPS dari Indocenter ini adalah setup-setup username dan password linux baru. Sebenarnya buat VPS dari Indocenter ini sudah ready to use ya, kita bisa langsung gunain buat server sesuai kebutuhan kita.
Web E-Voting yang aku host dengan Web Server
Setelah itu, aku coba install FTP Server untuk VPS ini biar transfer file antara local dan server jadi lebih mudah. Setelah itu aku cobain juga install Web Server di VPS ini. Disini aku pakai XAMPP buat stack servernya. Kebetulan Web Server disini aku gunain buat hosting web E-Voting Pemilihan Ketua Jurusan di Kampusku. Buat kebutuhan Web E-Voting, VPS ini gak ada kendala sama sekali, webku lancar banget buat diakses. Yeay!
Setelah utak-atik beberapa kali buat percobaan macam-macam server, aku jadi kepikiran ide gimana kalau aku buat hosting server game Minecraft ya, apa bisa? Hmmm. Lalu, aku cobain deh buat install server Minecraft di VPS ini dan ternyata bisa dan lancar jaya buat Mabar Minecraft, asyik.
Game Minecraft
So, overall pengalamanku menggunakan VPS dari Indocenter ini selama 3 bulan ini cukup memuaskan. Jadi aku juga mau ngucapin terima kasih untuk Indocenter dan KLAS atas kesempatannya.
